ZX Draw

Editor simple inspirado en ZX Spectrum para crear y editar gráficos compatibles con formatos clásicos (.zxp, .scr). Ideal para pixel art y assets para demos y juegos retro.

Resumen de funcionalidades

  • Apertura y guardado: soporte nativo para .zxp (formato del editor) y .scr (volcado de pantalla del ZX Spectrum).
  • Importar imágenes: arrastra o importa PNG/JPG/BMP y convierte automáticamente a paleta Spectrum.
  • Conversión con previsualización: modal interactivo donde ajustar Brillo, Contraste y Saturación mientras ves la conversión en tiempo real.
  • Edición por bloques: lienzo basado en bloques de 8×8, atributos de tinta/papel/bright/flash por bloque.
  • Herramientas: lápiz, selección por bloques, texto, invertir píxeles/atributos, voltear horizontal/vertical.
  • Animación: copiar áreas para previsualizar frames y reproducir secuencias (visor de animación).
  • Exportar: exporta a PNG para compartir o a .zxp para seguir editando.
  • Atajos: soporte para atajos comunes (Ctrl+S, Ctrl+O, S, P, T, etc.).

Flujo típico

  1. Importa una imagen o abre una .scr/.zxp.
  2. Usa el modal de importación para ajustar brillo/contraste/saturación y obtener la mejor conversión posible.
  3. Refina píxel a píxel, usa la selección y el portapapeles para montar sprites o tiles.
  4. Guarda como .zxp o exporta PNG. Para distribución en Windows, usa el instalador incluido.

Formato .scr

El soporte .scr interpreta los 6912 bytes de la pantalla Spectrum (6144 bitmap + 768 atributos) respetando el layout de memoria original.
