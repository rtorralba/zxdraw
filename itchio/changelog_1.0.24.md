# ZXDraw: Mejoras de usabilidad, Modo Oscuro e Importador de Boriel Basic · v1.0.24

---

## [ES] Español

### Resumen
Esta actualización se centra en mejorar la usabilidad, el manejo de archivos y la experiencia general del usuario. Se ha publicado la **versión 1.0.24** con todas las mejoras detalladas a continuación.

### Novedades y Mejoras

1. **Exportación y Guardado directo de archivos .scr**
   Ahora es mucho más fácil trabajar con pantallas estándar de ZX Spectrum (`.scr`):
   - Al guardar un archivo abierto como `.scr`, se sobrescribirá directamente sin preguntar.
   - En el menú "Guardar como", puedes elegir entre formato `.zxp` (nativo) o `.scr` (binario estándar).

2. **Opción "Usar atributos actuales"**
   Añadida una nueva opción en la barra de herramientas:
   - Al activarla, el pincel respetará los colores que ya existan en el lienzo, permitiéndote retocar solo los píxeles sin alterar el colorido.
   - Al desactivarla, los selectores de colores volverán a funcionar de forma normal.

3. **Modo Oscuro Forzado**
   La aplicación ahora utiliza siempre el modo oscuro para las barras de título, menús y barras de desplazamiento, garantizando una experiencia visual consistente.

4. **Zoom Inteligente**
   El zoom con la rueda del ratón ahora se centra automáticamente en la posición del puntero.

5. **Aviso de Cambios sin Guardar**
   Se ha añadido un diálogo de confirmación si intentas cerrar la aplicación con cambios pendientes de guardar.

6. **Importador de Boriel Basic (PutChars)**
   Nueva opción en el menú **Archivo > Importar**:
   - Permite recuperar archivos de Boriel Basic creados anteriormente, reconstruyendo los gráficos y atributos automáticamente en el editor.

## [EN] English

### Summary
This update focuses on improving usability, file handling, and the overall user experience. **Version 1.0.24** has been released with the following improvements.

### Features and Improvements

1. **Direct Saving and Export for .scr Files**
   Working with standard ZX Spectrum screen files (`.scr`) is now much easier:
   - Saving a file opened as `.scr` will now overwrite it directly without prompting.
   - You can now choose between `.zxp` (native) or `.scr` (standard binary) in the "Save As" menu.

2. **"Use Current Attributes" Toggle**
   A new tool has been added to the toolbar:
   - When active, the brush will respect existing colors on the canvas, allowing you to edit pixels without changing the ink or paper colors.
   - When deactivated, color selectors will function normally again.

3. **Forced Dark Mode**
   The application now always uses dark mode for title bars, menus, and scrollbars, regardless of your operating system settings.

4. **Smart Zoom**
   Mouse wheel zoom now automatically centers on the pointer position.

5. **Unsaved Changes Warning**
   A confirmation dialog has been added when trying to close the application with unsaved work.

6. **Boriel Basic (PutChars) Importer**
   New option in the **File > Import** menu:
   - Allows importing legacy Boriel Basic files, automatically reconstructing graphics and attributes in the editor.

## [PT] Português

### Resumo
Esta atualização foca na melhoria da usabilidade, manipulação de arquivos e na experiência geral do usuário. A **versão 1.0.23** foi lançada com as seguintes melhorias.

### Novidades e Melhorias

1. **Exportação e Salvamento direto de arquivos .scr**
   Agora é muito mais fácil trabalhar com telas padrão do ZX Spectrum (`.scr`):
   - Ao salvar um arquivo aberto como `.scr`, ele será sobrescrito diretamente sem perguntas.
   - No menu "Salvar como", você pode escolher entre o formato `.zxp` (nativo) ou `.scr` (binário padrão).

2. **Opção "Usar atributos atuais"**
   Nova opção adicionada na barra de ferramentas:
   - Quando ativada, o pincel respeitará as cores já existentes na tela, permitindo retocar apenas os pixels sem alterar o colorido.
   - Quando desativada, os seletores de cores voltarão a funcionar normalmente.

3. **Modo Escuro Forçado**
   O aplicativo agora utiliza sempre o modo escuro para barras de título, menus e barras de rolagem, garantindo uma experiência visual consistente.

4. **Zoom Inteligente**
   O zoom com a roda do mouse agora foca automaticamente na posição do ponteiro.

5. **Aviso de Alterações não Salvas**
   Adicionado um diálogo de confirmação se você tentar fechar o aplicativo com alterações pendentes.

6. **Importador de Boriel Basic (PutChars)**
   Nova opção no menu **Arquivo > Importar**:
   - Permite recuperar arquivos Boriel Basic criados anteriormente, reconstruindo os gráficos e atributos automaticamente no editor.
